Detailed Description

template<typename T>
class vex::gstencil< T >

Generalized stencil.

This is generalized stencil class. Basically, this is a small dense matrix. Generalized stencil may be used in combination with builtin or user functions of one argument, as in

vex::gstencil<double> S(ctx.queue(), 2, 3, 1, {
1, -1, 0,
0, 1, -1
});
y = sin(x * S);

This notation corresponds to , where c is center of the stencil. Please see github wiki for further examples of this class usage.
